# Exp149/150 Upload Scope

This directory is a snapshot of the files generated for the Exp149/150 virtual wafer-run conformal policy experiments.

- The Python implementation and its unit-test source are in `experiments/exp149_150/`.
- The CSV, Markdown, manifest, blocker, and checkpoint files here are copied from the experiment output directory.
- No raw database or raw time-series dataset is included.
- This upload does not perform or report an independent evaluation. It preserves the generated artifacts for handoff and review.
- The original local experiment files were not modified or overwritten by this upload.
