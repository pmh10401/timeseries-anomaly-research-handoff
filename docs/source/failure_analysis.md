# 이상치 탐지 실패 데이터셋 분석 보고서 (Failure Analysis)

본 보고서는 947개 데이터셋 중 모든 모델과 임계치 설정 기법에서 F1-Score `0.0`을 기록하며 탐지에 완전히 실패한 **130개 데이터셋 (13.73%)**의 메타데이터 및 물리적 특징을 통계적으로 비교 분석하여 모델의 맹점(Blind Spots)을 규명합니다.

| 통계 지표 | 성공군 (F1 > 0) | 실패군 (F1 = 0) | 분석 및 특이사항 |

| **중간값 시계열 길이** | 512.0 | 319.0 | 중간값 역시 실패군이 유의미하게 짧음 (정보량 부족) |

## 2. 실패 요인의 주요 학술적 가설 및 고찰

### 🚨 가설 1: 테스트 세트 내 이상치 절대 부족 문제 (극단적 불균형에 의한 F1 왜곡)
- **현상**: 실패군의 실제 테스트 이상치 개수의 중간값이 **1.0개**에 불과합니다. (많은 데이터셋이 단 1개의 이상치만 테스트 세트에 포함하고 있음)
- **통계적 맹점**: 이상치가 단 1개만 존재하는 경우, 모델이 98% 분위수 컷오프(테스트 샘플 100개 중 2개를 이상치로 판단)를 가동하면 최소 2개 이상의 샘플을 양성(Positive)으로 분류하게 됩니다. 이때 모델이 예측한 2개 중 실제 1개의 이상치가 포함되더라도 오탐(False Positive)이 1개 이상 무조건 발생하여 정밀도(Precision)와 재현율(Recall) 조율 시 수학적으로 매우 불안정한 F1 값을 갖게 되며, 만약 단 1개의 이상치를 간발의 차로 놓치면 재현율이 0%가 되어 F1-Score가 무조건 **0.0**으로 수직 하락합니다.

### 🚨 가설 2: 시계열 물리적 차원의 한계 (Short sequence length)
- **현상**: 실패군의 중간값 시계열 길이가 **96**차원으로, 성공군의 **170**차원 대비 약 1.8배 가까이 짧습니다.
- **원인**: 시계열 파형의 길이가 너무 짧은 경우(예: 15~40차원), 1D Convolutional 인코더와 디코더의 합성곱 필터(kernel_size=15)가 경계선 패딩에 가로막혀 시간적 특징을 압축하고 복원할 정보량 자체가 절대적으로 부족해집니다. 이로 인해 정상 데이터와 이상 데이터를 구분할 변별 점수 마진(Anomaly score margin)이 좁혀져 0.0의 F1을 기록하게 됩니다.
